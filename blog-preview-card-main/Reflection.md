What challenges did you face when refactoring your code to use Bootstrap?

One of the main challenges I faced was debugging layout issues, especially related to padding and margin. Sometimes it was difficult to identify where spacing was being applied incorrectly when using Bootstrap utility classes. I had to spend time reviewing the documentation and experimenting with different classes to properly adjust the layout and fix the spacing issues.

How did using Bootstrap utility classes and components simplify your styling process?

Using Bootstrap utility classes and components  simplified the styling process. It saved a lot of time because many styling properties are already built into Bootstrap. Instead of writing extensive CSS in a separate style.css file, I could directly apply Bootstrap classes to achieve the desired layout, spacing, and design. This made development faster and the code easier to maintain.

In what scenarios might you choose not to use Bootstrap and write custom CSS instead?

I would use bootstrap where the project requirement are bigger that would same time in writiing custom css.
